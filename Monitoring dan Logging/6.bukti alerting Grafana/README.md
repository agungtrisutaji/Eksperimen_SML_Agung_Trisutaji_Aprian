# Bukti Alerting Grafana

Tambahkan screenshot asli alerting Grafana secara manual.

Nama bukti yang disarankan:

```text
1.rules_latency.png
2.notifikasi_latency.png
3.rules_error.png
4.notifikasi_error.png
5.rules_model_loaded.png
6.notifikasi_model_loaded.png
```

Untuk target Advanced, siapkan minimal 3 alert rule, misalnya:

- high prediction latency;
- high prediction error count;
- model loaded status down.

Jangan membuat screenshot palsu.
